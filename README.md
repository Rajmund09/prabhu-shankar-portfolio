# Prabhu Shankar Mund - Portfolio

![Portfolio Preview](ppc.jpg)

A modern, responsive portfolio website showcasing skills, projects, and experience as an AI/ML Engineer & Full Stack Developer.

## 🌟 Features

- **Modern UI/UX**: Clean, dark theme with gradient accents
- **Responsive Design**: Works perfectly on all devices
- **Interactive Elements**:
  - 3D animated profile card
  - Skills toggle section
  - Project showcase with live demos
  - Contact form with validation
- **Performance Optimized**: Fast loading with smooth animations
- **SEO Friendly**: Proper meta tags and semantic HTML

## 🚀 Live Demo

[View Live Portfolio](https://prabhu-shankar-portfolio.vercel.app/)

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Animations**: GSAP, ScrollTrigger
- **Icons**: Font Awesome 6
- **Fonts**: Google Fonts (Montserrat, Inter, Press Start 2P)
- **3D Graphics**: Three.js (in skills showcase)
